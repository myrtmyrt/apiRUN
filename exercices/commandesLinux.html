Les 10 commandes Linux les plus importantes

  1. Lister les Fichiers et Répertoires dans le Répertoire Actuel : ls
  2. Changer de Répertoire : cd chemin/vers/le/dossier
  3. Copier un Fichier : cp source destination
  4. Déplacer ou Renommer un Fichier : mv ancien_nom nouveau_nom
  5. Afficher le Contenu d'un Fichier : cat nom_du_fichier
  6. Modifier un Fichier avec l'Éditeur de Texte Nano : nano nom_du_fichier
  7. Supprimer un Fichier : rm nom_du_fichier
  8. Afficher l'Adresse IP de la Machine : ip addr show
  9. Installer un Paquet : sudo apt-get install nom_du_paquet
  10. Redémarrer la Machine : sudo reboot
